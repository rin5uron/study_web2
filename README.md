### 📚 基本情報技術者試験学習　[fe-study](https://github.com/rin5uron/fe-study)

基本情報技術者試験の学習ログと問題演習をGitHub Issuesで管理。ITパスポート取得済みから基本情報技術者試験合格を目指す学習記録。

**主な特徴**:
- GitHub Issuesを活用した学習進捗管理
- 実践的に学んだ記録
