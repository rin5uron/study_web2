JavaScript II - ループ練習

この学習では、JavaScriptのループ処理を学びます。while、for、fizzbuzzなど、繰り返し処理の基本を理解するトレーニングです。

学んだこと

  1. while - 条件が真の間ループする
  2. for - 回数を指定してループする
  3. fizzbuzz - 条件分岐を使った有名な問題

1. while文

▼

条件の間だけループする。条件や繰り返し処理の指定を忘れると無限ループになるので注意。

1〜100までの数を表示する

let i = 1;
while(i <= 100){
    console.log(i);
    i++;
    // i++ は i += 1（i=i+1）の省略形
}

2. for文

▼

for(初期化/変数定義; 条件; 増減/繰り返し処理)

1〜100までの数を表示する

for (let number = 1; number<=100; number++){
    console.log(number);
}

練習問題

① 偶数抽出

▼
1〜50までの偶数だけをコンソールに表示してください
ヒント: i % 2 === 0 で偶数判定

あなたの回答

let num1 = 1;
while(num1 <= 50){
    if(num1 % 2 === 0){
        console.log(num1);
    }
    num1++;
}
○ 正解

⑤ 条件付き合計

▼
1から順番に数を足していき、合計が100を超えたらループを終了してください
最後に「合計: ○○」「何個足した: ○個」を表示してください
ヒント: breakを使うか、条件を工夫する

あなたの回答

let num5 = 1;
let sum5 = 0;
while(num5 <= 100){
    sum5 = sum5 + num5;
    num5++;
    if(sum5 > 100){
        break;
    }
}
console.log("合計: " + sum5);
console.log("何個足した: " + num5 + "個");
○ 正解

⑥ FizzBuzz

▼
1〜100までの数を表示してください。ただし：
・3の倍数の時は「Fizz」
・5の倍数の時は「Buzz」
・15の倍数の時は「FizzBuzz」
を表示してください

あなたの回答

for(let number = 1; number<=100; number++){
    if(number % 15 === 0){
        console.log("fizzbuzz");
    }else if(number % 3 === 0){
        console.log("fizz");
    }else if(number % 5 === 0){
        console.log("buzz");
    }else{
        console.log(number);
    }
}
○ 正解
