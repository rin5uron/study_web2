1. .valueプロパティ

      
入力欄に入力されたテキストの中身を取り出すプロパティ

      const name = document.getElementById('name').value;
      
→ input要素の値を取得できる

    


    

      
2. JSON.stringify()

      
オブジェクトをJSON文字列に変換できる

      const jsonText = JSON.stringify(data);
      
→ JavaScriptオブジェクトを文字列に変換してlocalStorageに保存可能

    


    

      
3. null（ぬる）

      
意図的に無効・未設定であることを示す値

      JSON.stringify(obj, null, 2)
      
→ 第2引数にnullを渡し、第3引数に2を渡すと、オブジェクトの各行の冒頭に2文字分のインデントが入る

      
例：見やすい形式で表示される
